# Calculadora BaZi (WebApp)

Este proyecto es una calculadora BaZi simple que genera los Cuatro Pilares (Año, Mes, Día, Hora) e interpreta el Día Maestro y las deidades básicas. Incluye exportación a PDF.

## Archivos incluidos

- `index.html`: Página principal
- `style.css`: Estilos visuales
- `script.js`: Lógica de cálculo BaZi
- `html2pdf.bundle.min.js`: Librería necesaria para exportar a PDF

## Cómo usar

1. Iniciá sesión en GitHub y creá un repositorio nuevo (sin README).
2. Subí estos 4 archivos directamente al repositorio.
3. En **Settings > Pages**, seleccioná `branch: main` y `/root` como carpeta.
4. GitHub generará un enlace público como:

   `https://tu_usuario.github.io/tu_repositorio/`

¡Y listo! La app estará disponible online y podrás compartirla.
